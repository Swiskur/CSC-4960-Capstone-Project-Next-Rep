# CSC-4960-Capstone-Project-Next-Rep

# Overview
Next-Rep is Madonna University Athletic Department's scheduling website for students to connect with sports trainers. The student must register on the site with their Madonna E-mail to access the trainers and their calendars with appointment availabilities. Trainers access the site to populate their availability on their personal calendar by filling in the time blocks on a given date. When a timeslot has been booked, reminders are sent out with the message, “Your upcoming appointment, trainer, status, and notes.” If there are no appointments available for the trainer, the message “No Available Appointments” will be visible on the screen. Appointments can be changed or cancelled.

The changes made to the scheduling process will result in a more efficient system, resolving communication issues between the trainers and athletes. The ability to promptly see a trainer as needed will result in fewer injuries, enhancing overall team performance.
